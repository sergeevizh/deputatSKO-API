Сервер закрыт basic authorization (admin, 123)

1)      depsko.my/api/create_user.php – Создает пользователя в БД.

Принимает:

{

  "firstname" : "Paul",

  "lastname" : "Starkov",

  "thirdname" : "Anatolevich",

  "email" : "paul.starkov@gmail.com",

  "password" : "123",

  "phone" : "77772806421",

  "adress_id" : 3

}

Ответ:

{

    "error": 0, - 1 если есть ошибка

    "message": "Пользователь был создан.",

    "sms": 1 – значить что СМС была отправлена на номер, -1-не отравлена

}

2)      depsko.my/api/activate_user.php – Активирует созданного пользователя, если не прошел активацию то невозможно будет авторизоваться

Принимает

{

  "phone":"77772806421",

  "code":1231 – приходит в СМС

}

Ответ:

{

    "error": 0,

    "message": "Пользователь был активирован."

}
